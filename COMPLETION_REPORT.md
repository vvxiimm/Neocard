## 🚀 КАК ЗАПУСТИТЬ

### 1. Установить зависимости
```bash
pnpm install
```

### 2. Запустить базы данных
```bash
docker-compose up -d postgres redis
```

### 3. Настроить БД и добавить карты
```bash
pnpm db:generate
pnpm db:push
pnpm --filter @nexus/api db:seed
```

### 4. Запустить серверы
```bash
# Terminal 1 - Backend
pnpm --filter @nexus/api dev

# Terminal 2 - Frontend
pnpm --filter @nexus/web dev
```

### 5. Открыть игру
- Frontend: http://localhost:3000
- Backend: http://localhost:3001
